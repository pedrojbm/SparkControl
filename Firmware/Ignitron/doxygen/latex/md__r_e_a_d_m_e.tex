

 

An ESP32 based foot pedal to communicate with the Spark Amp and App via Bluetooth LE.

{\bfseries{Ignitron}} gives you full control over your Spark Amp\+:
\begin{DoxyItemize}
\item Switch between the four {\bfseries{hardware presets}}
\item Switch between {\bfseries{custom saved presets}} organized in banks
\item {\bfseries{Activate / Deactivate effects}} for the selected preset
\item {\bfseries{Load new presets}} via the Spark App
\item {\bfseries{Delete}} stored presets
\end{DoxyItemize}

Adding new presets to {\bfseries{Ignitron}} can easily be done as it can also act as a Spark Amp. Simply connect to {\bfseries{Ignitron}} with your Spark app and load new presets directly from Tone\+Cloud (or your downloaded presets) to {\bfseries{Ignitron}}.

One big advantage of {\bfseries{Ignitron}} is that it communicates with your Spark Amp using the {\bfseries{Bluetooth LE}} protocol. This means you can still connect your mobile with your Spark Amp as an audio speaker and play along your favorite songs while you control your effects.

The current active preset and effects are indicated via LEDs.\textbackslash{} In addition, the {\bfseries{built-\/in display}} provides information on
\begin{DoxyItemize}
\item selected bank and preset number
\item selected preset name
\item activated pedal types in the FX chain
\item connection status to Spark Amp and Spark App

{\bfseries{Ignitron}} has two operation modes, {\bfseries{APP mode}} and {\bfseries{AMP mode}}.\textbackslash{} In {\bfseries{APP mode}} (default mode on startup), {\bfseries{Ignitron}} connects to a Spark Amp and behaves like the Spark App towards the Spark Amp. It can then be used to switch between saved presets and/or toggle FX swtiches. {\bfseries{AMP mode}} can be used to manage the presets stored on {\bfseries{Ignitron}}, presets can be added from the app or be deleted from {\bfseries{Ignitron}}.\textbackslash{} To enter {\bfseries{AMP mode}}, hold the {\ttfamily \mbox{\hyperlink{struct_preset}{Preset}} 1} button during startup.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{APP mode}\label{md__r_e_a_d_m_e_autotoc_md2}
In APP mode, the foot switches can be used to either switch between pre-\/saved presets ({\bfseries{\mbox{\hyperlink{struct_preset}{Preset}} mode}}) or to control all single effects in the selected preset ({\bfseries{FX mode}}). You can easily toggle between these modes by long pressing the {\ttfamily Bank-\/\+Up} button. When selecting {\bfseries{\mbox{\hyperlink{struct_preset}{Preset}} mode}}, four buttons are used to select presets the other two buttons are used to navigate through different preset banks. This way the user has access to a huge number of saved presets. When pressing the foot switch of the current active preset, the effect configured in the DRIVE section can be enabled and disabled. In {\bfseries{FX mode}}, the user has direct access to all effects of the selected preset. \hypertarget{md__r_e_a_d_m_e_autotoc_md7}{}\doxysubsection{AMP mode}\label{md__r_e_a_d_m_e_autotoc_md7}
In AMP mode, {\bfseries{Ignitron}} acts like a Spark AMP and can communicate with the Spark app running on a mobile. New presets can be stored on {\bfseries{Ignitron}} and existing presets can be deleted.\hypertarget{md__r_e_a_d_m_e_autotoc_md8}{}\doxysubsubsection{Connecting the Spark app with $<$strong$>$\+Ignitron$<$/strong$>$}\label{md__r_e_a_d_m_e_autotoc_md8}

\begin{DoxyEnumerate}
\item Start {\bfseries{Ignitron}} in AMP mode (hold {\ttfamily \mbox{\hyperlink{struct_preset}{Preset}} 1} button during startup).
\item Open the Spark App on the mobile
\item For the first connection, make sure to have your real Spark Amp powered off to avoid the app conneting to it.
\item Hit the connect button in the app (or the + button in the connection overview)
\item Once the connection is established, you can give the connection a name in the app so you can distinguish from your regular Spark Amp connection.
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md9}{}\doxysubsubsection{Storing a new preset (a preset from the app needs to be loaded)}\label{md__r_e_a_d_m_e_autotoc_md9}

\begin{DoxyEnumerate}
\item Start {\bfseries{Ignitron}} in AMP mode (hold \mbox{\hyperlink{struct_preset}{Preset}} 1 button during startup).
\item Connect the Spark App with {\bfseries{Ignitron}} (see above)
\item Select a preset in the app (either saved in the app or from the Tone\+Cloud).
\item The preset name should appear in the bottom line of the display
\item The {\ttfamily \mbox{\hyperlink{struct_preset}{Preset}}}/{\ttfamily Bank} buttons on {\bfseries{Ignitron}} can be used to navigate to the desired preset position.
\item Press the {\ttfamily \mbox{\hyperlink{struct_preset}{Preset}}} button to mark the position for storing
\item The LED of the selected preset position should start blinking
\item Hit the same {\ttfamily \mbox{\hyperlink{struct_preset}{Preset}}} button a second time to confirm storage (Hitting any other {\ttfamily \mbox{\hyperlink{struct_preset}{Preset}}}/{\ttfamily Bank} button will revert the state back to navigating)
\item The preset will be saved to the selected position, other presets will be pushed back

\hypertarget{md__r_e_a_d_m_e_autotoc_md14}{}\doxysubsubsection{Manually creating presets}\label{md__r_e_a_d_m_e_autotoc_md14}
{\itshape This method is only recommended if a preset cannot be transferred via the Spark app or if the preset files have been received/generated in a different way.}
\begin{DoxyEnumerate}
\item Create a {\bfseries{preset file}} in JSON format as shown above
\item Make sure the {\bfseries{file name}} does not exceed 31 characters (including the {\ttfamily .json} suffix)
\item Put the file name into the {\bfseries{data folder}}
\item Insert the file name (including the {\ttfamily .json} suffix) to the desired prefix location in the {\ttfamily Preset\+List.\+txt} file
\item {\itshape Place the Preset\+List.\+txt file into the data folder (if not already there)}
\item {\bfseries{Upload the data folder}} to {\bfseries{Ignitron}} via the Arduino IDE (or other tools)
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md15}{}\doxyparagraph{Preset format}\label{md__r_e_a_d_m_e_autotoc_md15}
{\bfseries{Ignitron}} stores presets in a JSON format using the SPIFFS file system. Each preset is stored in a separate file and presets are organized in a separate text file called \textquotesingle{}Preset\+List.\+txt\textquotesingle{}.
